# Happy Birthday Prachu — Website

Small multi-page static website to wish Prachu a happy birthday (24 Dec 2025). Replace photos and songs with real files.

Files:
- [index.html](index.html) — Welcome
- [gallery.html](gallery.html) — Replace images in assets/photos/
- [songs.html](songs.html) — Play or upload songs
- [celebration.html](celebration.html) — Cake, candles, balloons, confetti

Where to put your assets:
- Photos: `assets/photos/` (replace the SVGs or add JPG/PNG with same filenames)
- Music: you can upload via the Songs page or put files in `assets/music/` and update `songs.html`.

Run locally (quick methods):

Double-click `index.html` to open in your browser, or run a simple local server (recommended):

```bash
# from project folder
python -m http.server 8000
# then open http://localhost:8000
```

Notes:
- Browsers may block autoplay of audio; click a button to start sound.
- Feel free to ask me to add more animations, customize colors, or incorporate specific photos/songs.
